This demo allows you to run ANY javascript file in a web worker!

Wanna try?

Hit the Upload Script button to upload a javascript file that contains the script which you would like to run in the worker.

Hit the Start Script button to start the execution of your script. This will call the start() function in your script (if there is one).

If your script posts any messages via the postMessage() API to the main script on this page, then it will be displayed as Incoming value.

You can hit the Stop Script button to stop the execution of your script. This will call the stop() function in your script (if there is one) and terminate the worker.
